Assim que rodar, o programa tem um menu intuitivo.

Tenha em mente o seguinte fluxo descrito em "Plano.png"

Um podcast é uma cabeça de uma lista de episódios 
Um episódio é uma cabeça de uma lista de palavras
Uma playlist é uma cabeça de uma lista de episódios

Apenas a lista de palavras é simplesmente ligada. Todas as outras são duplamente ligadas.
